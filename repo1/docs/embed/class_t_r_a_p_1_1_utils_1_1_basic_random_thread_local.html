Detailed Description

template<typename engine, typename Seeder = SeederDefault, template< typename > class IntegerDist = std::uniform_int_distribution, template< typename > class RealDist = std::uniform_real_distribution, typename BoolDist = std::bernoulli_distribution>
class TRAP::Utils::BasicRandomThreadLocal< engine, Seeder, IntegerDist, RealDist, BoolDist >

Base template class for random with thread_local API and thread_local internal member storage.

Note: It IS thread safe but less efficient then BasicRandomStatic.

Template Parameters
engineA random engine with interface like in the std::mt19937.
SeederA seeder type which return seed for internal engine through operator().
