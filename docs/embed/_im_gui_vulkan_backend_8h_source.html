30 // dear imgui: Renderer Backend for Vulkan
31 // This needs to be used along with a Platform Backend (e.g. GLFW, SDL, Win32, custom..)
32 
33 // Implemented features:
34 // [X] Renderer: Support for large meshes (64k+ vertices) with 16-bit indices.
35 // Missing features:
36 // [ ] Renderer: User texture binding. Changes of ImTextureID aren't supported by this backend! See https://github.com/ocornut/imgui/pull/914
37 
38 // You can use unmodified imgui_impl_* files in your project. See examples/ folder for examples of using this.
39 // Prefer including the entire imgui/ repository into your project (either as a copy or as a submodule), and only build the backends you need.
40 // If you are new to Dear ImGui, read documentation from the docs/ folder + read the top of imgui.cpp.
41 // Read online: https://github.com/ocornut/imgui/tree/master/docs
42 
43 // The aim of imgui_impl_vulkan.h/.cpp is to be usable in your engine without any modification.
44 // IF YOU FEEL YOU NEED TO MAKE ANY CHANGE TO THIS CODE, please share them and your feedback at https://github.com/ocornut/imgui/
45 
46 // Important note to the reader who wish to integrate imgui_impl_vulkan.cpp/.h in their own engine/app.
47 // - Common ImGui_ImplVulkan_XXX functions and structures are used to interface with imgui_impl_vulkan.cpp/.h.
48 // You will use those if you want to use this rendering backend in your engine/app.
49 // - Helper ImGui_ImplVulkanH_XXX functions and structures are only used by this example (main.cpp) and by
50 // the backend itself (imgui_impl_vulkan.cpp), but should PROBABLY NOT be used by your own engine/app code.
51 // Read comments in imgui_impl_vulkan.h.

142  VkSurfaceKHR Surface;
