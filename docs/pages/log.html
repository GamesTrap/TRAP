Log¶

Quick Access¶

TP_TRACE¶

template<typename... Args>
constexpr void TP_TRACE(const Args& ... args)
Description
Does the same as TRAP::Log::Trace

TP_DEBUG¶

template<typename... Args>
constexpr void TP_DEBUG(const Args& ... args)
Description
Does the same as TRAP::Log::Debug

Note

Debug messages are disabled for Release builds!

Level¶

enum class TRAP::Log::Level
{
   Trace,
   Debug,
   Info,
   Warn,
   Error,
   Critical
};
Description
TRAP::Log::Level is used to set the importance of a message.

Constructor¶

TRAP::Log();
Description
Creates an instance of TRAP::Log.

Destructor¶

TRAP::~Log();
Description
Destructs TRAP::Log and saves every message in the internal buffer.

Trace¶

template<typename... Args>
static void TRAP::Log::Trace(Args&& ... args);
Description
Creates a message from the variadic list with TRAP::Log::Level::Trace.
Parameters

  • args Specifies a variadic list that gets processed to a message and then used by TRAP::Log.
