Windows¶

General¶

Windows in TRAP Engine are using the library GLFW

Note

TRAP Engine always creates 1 TRAP::Window and uses it as Main Window!
Properties of this TRAP::Window can be set in Engine.cfg

WindowProps¶

Window Properties
TRAP::WindowProps(std::string title,
                  const uint32_t width,
                  const uint32_t height,
                  const uint32_t refreshRate,
                  const uint32_t vsync,
                  const TRAP::Window::DisplayMode displayMode,
                  const uint32_t monitor,
                  const TRAP::Window::CursorMode cursorMode,
                  const bool rawMouseInput);
Description
Holds properties for a TRAP::Window.
Gets used on TRAP::Window Constructor.
Parameters

  • title Sets the Title of TRAP::WindowProps. Default value: "TRAP Engine"

  • width Sets the Width of TRAP::WindowProps. Default value: 1280

  • height Sets the Height of TRAP::WindowProps. Default value: 720

  • refreshRate Sets the Refresh Rate of TRAP::WindowProps. Default value: 60

  • vsync Sets the VSync interval of TRAP::WindowProps. Default value: 0

  • displayMode Sets the Display Mode of TRAP::WindowProps. Default value: TRAP::Window::DisplayMode::Windowed

  • monitor Sets the Monitor of TRAP::WindowProps. Default value: 0

  • cursorMode Sets the TRAP::Window::CursorMode. Default value: TRAP::Window::CursorMode::Normal

  • rawMouseInput Sets the Mouse Input. Default value: false

Note

Width, Height, RefreshRate, VSync, TRAP::Window::DisplayMode and Monitor values loaded by Engine.cfg only affect the Main Window!

Note

If Raw Mouse Input is unsupported then rawMouseInput will be set to false!

Note

If displayMode is set to TRAP::Window::DisplayMode::Borderless then width, height and refreshRate will be ignored! The Native Resolution and Refresh Rate of monitor will be used!

Warning

If displayMode is either set to TRAP::Window::DisplayMode::Borderless or TRAP::Window::DisplayMode::Fullscreen and another TRAP::Window with TRAP::Window::DisplayMode::Borderless or TRAP::Window::DisplayMode::Fullscreen already uses the same monitor then displayMode will be set to TRAP::Window::DisplayMode::Windowed!

If width, height and refreshRate are set to 0 then width will be set to 800, height will be set to 600 and refreshRate will be set to the monitor’s Native Refresh Rate!

SetDisplayMode¶

void TRAP::Window::SetDisplayMode(const TRAP::Window::DisplayMode& mode,
                                  uint32_t width,
                                  uint32_t height,
                                  uint32_t refreshRate);
Description
Sets the TRAP::Window::DisplayMode of a TRAP::Window

Note

This can also be used to Resize a TRAP::Window

Parameters

  • mode Specifies the new TRAP::Window::DisplayMode of a TRAP::Window

  • width Specifies the new Width of a TRAP::Window. Default value: 0

  • height Specifies the new Height of a TRAP::Window. Default value: 0

  • refreshRate Specifies the new Refresh Rate of a TRAP::Window. Default value: 0

Note

If mode is TRAP::Window::DisplayMode::Windowed & width and height are 0 then width, height and refreshRate will be set to the last used values of the TRAP::Window

Note

If mode is TRAP::Window::DisplayMode::Borderless then width, height and refreshRate will be ignored! The Native Resolution of the current Monitor of the TRAP::Window will be used!

Note

If mode is TRAP::Window::DisplayMode::Fullscreen & width, height and refreshRate are 0 then the Native Resolution and Refresh Rate of the current Monitor of the TRAP::Window will be used!

Note

refreshRate only affects the TRAP::Window if mode is TRAP::Window::DisplayMode::Fullscreen!

Warning

If mode is either set to TRAP::Window::DisplayMode::Borderless or TRAP::Window::DisplayMode::Fullscreen and another TRAP::Window with TRAP::Window::DisplayMode::Borderless or TRAP::Window::DisplayMode::Fullscreen already uses the same monitor then nothing will be changed!
