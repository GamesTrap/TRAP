Image¶

General¶

Note

TRAP Engine currently supports the following image formats: PNG, BMP, PGM, PPM, PNM, PAM, PFM, TGA and custom images loaded from memory

Warning

Image formats that will be added in the future: JPG, EXR, HDR and TIFF

ColorFormat¶

enum class TRAP::Image::ColorFormat
{
    NONE = 0,
    RGB,
    RGBA,
    GrayScale,
    GrayScaleAlpha
};
Description
TRAP::Image::ColorFormat is used to specify the color format of a TRAP::Image.

LoadFromMemory¶

static TRAP::Scope<TRAP::Image> TRAP::Image::LoadFromMemory(uint32_t width, uint32_t height, uint32_t bitsPerPixel, TRAP::Image::ColorFormat format, const std::vector<uint8_t>& pixelData);

static TRAP::Scope<TRAP::Image> TRAP::Image::LoadFromMemory(uint32_t width, uint32_t height, uint32_t bitsPerPixel, TRAP::Image::ColorFormat format, const std::vector<uint16_t>& pixelData);

static TRAP::Scope<TRAP::Image> TRAP::Image::LoadFromMemory(uint32_t width, uint32_t height, uint32_t bitsPerPixel, TRAP::Image::ColorFormat format, const std::vector<float>& pixelData);
Description
Creates a custom TRAP::Image from memory with the given parameters
Parameters

  • width Specifies the width for the TRAP::Image.

  • height Specifies the height for the TRAP::Image.

  • bitsPerPixel Specifies the bits per pixel for the TRAP::Image.

  • format Specifies the TRAP::Image::ColorFormat for the TRAP::Image.

  • pixelData Specifies the raw pixel data (from top left to bottom right) with the given color format for the TRAP::Image.

GetPixelData¶

const void* TRAP::Image::GetPixelData() const;
Description
Returns the raw pixel data as a readonly pointer to an array with the size of TRAP::Image::GetPixelDataSize and one of the follwoing types: uint8_t, uint16_t or float(HDR)

Note

This gets used by TRAP and shouldn’t be used/needed by the client!
