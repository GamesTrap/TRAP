Events¶

General¶

Warning

Events in TRAP Engine are currently blocking, meaning when an event occurs it immediately gets dispatched and must be dealt with right then and there!

Event¶

Description
TRAP::Events::Event is used as a base class for every event type in TRAP::Events::EventType

EventCategory¶

enum class TRAP::Events::EventCategory
{
    None        = 0,
    Application = BIT(0),
    Input       = BIT(1),
    Keyboard    = BIT(2),
    Mouse       = BIT(3),
    MouseButton = BIT(4),
    Controller  = BIT(5)
};
Description
TRAP::Events::EventCategory is used to specifiy the category of a TRAP::Events::Event

See also

BIT

EventDispatcher¶

Description
TRAP::Events::EventDispatcher is used to dispatch TRAP::Events::Events to different functions

Constructor¶

TRAP::Events::EventDispatcher(TRAP::Events::Event& event);
Description
Creates a new TRAP::Events::EventDispatcher for the given TRAP::Events::Event
Parameters
event Specifies the TRAP::Events::Event to dispatch

Dispatch¶

template<typename T, typename F>
bool TRAP::Events::EventDispatcher::Dispatch(const F& func);
Description
This function dispatches the specified TRAP::Events::Event class to the specified function func and returns true if dispatched successfully
Parameters
T Specifies the TRAP::Events::Event class to dispatch
F(func) Specifies the function to dispatch the TRAP::Events::Event to

See also

Use a lambda to call an user created function that takes the TRAP::Events::Event as a parameter Example: Dispatch<TRAP::Events::WindowCloseEvent>([this](TRAP::Events::WindowCloseEvent& e) {return OnWindowClose(e); });

EventType¶

enum class TRAP::Events::EventType
{
    None,
    WindowClose, WindowResize, WindowMinimize, WindowMaximize, WindowRestore, WindowFocus, WindowLostFocus, WindowMove, WindowDrop, WindowContentScale, FrameBufferResize
    AppTick, AppUpdate, AppRender,
    KeyPress, KeyRelease, KeyType,
    MouseButtonPress, MouseButtonRelease, MouseMove, MouseScroll, MouseEnter, MouseLeave,
    ControllerConnect, ControllerDisconnect,
    MonitorConnect, MonitorDisconnect
};
Description
TRAP::Events::EventType is used to specifiy the type of a TRAP::Events::Event

Warning

TRAP::Events::EventType::AppTick, TRAP::Events::EventType::AppUpdate and TRAP::Events::EventType::AppRender are currently unused by TRAP Engine!
