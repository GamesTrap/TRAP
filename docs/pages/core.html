TRAP_ENABLE_ASSERTS¶

TRAP_ENABLE_ASSERTS
Description
This macro specifies if asserts should be used or not

Warning

TRAP_ENABLE_ASSERTS only gets defined if TRAP Engine gets build in Debug or “Release with Debug info” mode!

TRAP_ASSERT¶

TRAP_ASSERT(x, ...)
Description
If x is false this will trigger a TP_ERROR with the message "Assertion Failed: + contents from ... + " @[__FILE__:__LINE__]" and then breaks the engine with TRAP_DEBUG_BREAK
Parameters

  • x Specifies the value to check

  • ... Specifies the message to print if the assertion gets triggered

Warning

This macro won’t do anything if TRAP_ENABLE_ASSERTS is not defined!
