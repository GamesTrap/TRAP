Monitors¶

VideoMode¶

struct TRAP::Monitor::VideoMode
{
    int32_t Width,
    int32_t Height,
    int32_t RefreshRate
};
Description
TRAP::Monitor::VideoMode is used to get the Width, Height and RefreshRate of a VideoMode that a Monitor can display. This is used by TRAP::Monitor.

Monitor¶

Constructor¶

explicit TRAP::Monitor(TRAP::INTERNAL::WindowingAPI::InternalMonitor* monitor);
Description
Creates a new TRAP::Monitor
Parameters

  • monitor Pointer to an INTERNAL::WindowingAPI::InternalMonitor for the new TRAP::Monitor.

Note

This Constructor gets used by TRAP and shouldn’t be used/needed by the client!

explicit TRAP::Monitor(uint32_t monitor);
Description
Creates a new TRAP::Monitor
Parameters

  • monitor Specifies the index into the Monitor list used by the new TRAP::Monitor. 0 is the Primary Monitor.

GetName¶

std::string TRAP::Monitor::GetName() const;
Description

GetVideoModes¶

std::vector<TRAP::Monitor::VideoMode> TRAP::Monitor::GetVideoModes() const;
Description
Returns a vector containing all TRAP::Monitor::VideoModes that the Monitor inside TRAP::Monitor can use.

GetCurrentVideoMode¶

TRAP::Monitor::VideoMode TRAP::MonitorGetCurrentVideoMode() const;
Description
Returns the currently used TRAP::Monitor::VideoMode from the Monitor inside TRAP::Monitor.

IsInUse¶

bool TRAP::Monitor::IsInUse() const;
Description
Returns true if the Monitor inside the TRAP::Monitor is used by a Fullscreen or Borderless TRAP:Window.

GetID¶

uint32_t TRAP::Monitor::GetID() const;
Description
Returns the index into the Monitor list from the Monitor inside the TRAP::Monitor.
0 is the Primary Monitor.

GetInternalMonitor¶

void* TRAP::Monitor::GetInternalMonitor() const;
Description
Returns a Pointer to the TRAP::INTERNAL::WindowingAPI::InternalMonitor used by the Monitor inside the TRAP::Monitor.

Note

This gets used by TRAP and shouldn’t be used/needed by the client!

GetAllMonitors¶
