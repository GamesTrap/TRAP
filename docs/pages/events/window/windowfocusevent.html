WindowFocusEvent¶

Note

TRAP::Events::WindowFocusEvent inherits from TRAP::Events::Event

Constructor¶

TRAP::Events::WindowFocusEvent(const std::string_view title);
Description
This is used to create new TRAP::Events::WindowFocusEvents
Parameters
title Specifies the title of the TRAP::Window the TRAP::Events::WindowFocusEvent is for

GetTitle¶

std::string_view TRAP::Events::WindowFocusEvent::GetTitle() const;
Description
Returns the title of the window from the TRAP::Events::WindowFocusEvent

GetStaticType¶

static TRAP::Events::EventType TRAP::Events::WindowFocusEvent::GetStaticType();
Description
Returns TRAP::Events::EventType::WindowFocus

GetEventType¶

TRAP::Events::EventType TRAP::Events::WindowFocusEvent::GetEventType() const override;
Description
Returns TRAP::Events::EventType::WindowFocus

GetName¶

const char* TRAP::Events::WindowFocusEvent::GetName() const override;
Description
Returns a c-string containing: "WindowFocus"

GetCategoryFlags¶

int32_t TRAP::Events::WindowFocusEvent::GetCategoryFlags() const override;
Description
Returns TRAP::Events::EventCategory::Application casted to an int32_t
