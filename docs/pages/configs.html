Configs¶

General¶

Note

Config files are EoL(CRLF/LF) independent

Engine.cfg¶

#TRAP Config
Width = 1280
Height = 720
RefreshRate = 144
VSync = 0
FPSLimit = 0
DisplayMode = Windowed
Maximized = FALSE
Monitor = 0
RenderAPI = OpenGL
ControllerAPI = XInput
Description
Default TRAP::Config file that always gets loaded/created
Parameters

  • Width Specifies the width for the TRAP::Window. This is ignored when using TRAP::Window::DisplayMode::Borderless!

  • Height Specifies the height for the TRAP::Window. This is ignored when using TRAP::Window::DisplayMode::Borderless!

  • RefreshRate Specifies the refresh rate for the TRAP::Window. This get used when using TRAP::Window::DisplayMode::Fullscreen!

  • VSync Specifies the VSync interval for the TRAP::Window.

  • FPSLimit Specifies a FPSLimit for TRAP. Valid Values: 0 & 25 - 500. Value 0 disables the FPSLimit.

  • DisplayMode Specifies the TRAP::Window::DisplayMode for the TRAP::Window.

  • Maximized Specifies if the TRAP::Window should be maximized. This get used when using TRAP::Window::DisplayMode::Windowed!

  • Monitor Specifies the TRAP::Monitor to use by the TRAP::Window. Value 0 is the Primary Monitor!

  • RenderAPI Specifies the RenderAPI to use by TRAP. Valid Values: OpenGL, Vulkan, D3D12(Windows only)!

  • ControllerAPI Specifies the ControllerAPI to use by TRAP. Valid Values: XInput(Windows only), DirectInput(Windows only), Linux(Linux only)!

  • filename Specifies the file name (and path) to load data from.

SaveToFile¶

bool TRAP::Utils::Config::SaveToFile(const std::string& filename);
Description
Saves/Updates the specified Config file with the current values of the TRAP::Utils::Config instance.
Parameters

  • filename Specifies the file name (and path) to save data in.

IsChanged¶

bool TRAP::Utils::Config::IsChanged() const;
Description
Returns true if the values in the TRAP::Utils::Config instance were changed since last loading.

Get¶

template<typename T>
void TRAP::Utils::Config::Get(const std::string& key, T& value) const;
Description
Returns the value converted to the type T found from the specified string.
Parameters

  • key Specifies the string to get data from.

  • value Specifies the variable to output the data converted to the type T found from the specified key.

Get¶

template<typename T>
void TRAP::Utils::Config::Get(const std::string& key, std::vector<T>& value) const;
Description
Returns the a vector containing the values converted to the type T found from the specified string.

  • key Specifies the string to get data from.

  • value Specifies the vector to output the data converted to the type T found from the specified key.

Set¶

template<typename T>
void TRAP::Utils::Config::Set(const std::string& key, const T value);
Description
Assigns the value of type T to the specified string.

  • key Specifies the string to set data to.

  • value Specifies the data to set for the specified key.

Set¶

template<typename T>
void TRAP::Utils::Config::Set(const std::string& key, const std::vector<T>& value);
Description
Assigns the values of the vector with type T to the specified string.

  • key Specifies the string to set data to.

  • value Specifies the vector data to set for the specified key.

void TRAP::Utils::Config::Print() const;
Description
Outputs every string with its assigned value. Last output is the total amount of every element in the loaded Config file.
